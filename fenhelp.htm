Analyse

Allows you to analyse a chess position. You can also start a game from the position in a new web page.

Optional URL parameters

fen=fen

Load the specified FEN as the position to analyse.

act=ana

Auto start an analysis.

Example:-

https://op12no2.github.io/lozza-ui/fen.htm?act=eva&fen=5k2/p7/3R4/2p3p1/1bP3P1/5N1P/3p1PK1/3r4

act=eva

Auto start an evaluation.

Example:-

https://op12no2.github.io/lozza-ui/fen.htm?act=eva&fen=5k2/p7/3R4/2p3p1/1bP3P1/5N1P/3p1PK1/3r4
