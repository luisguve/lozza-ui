Play UI

Play a game of chess against Lozza Chess. By default you will be playing with the white pieces. Use the Play menu to change colour and start a new game.

You can change the strength of Lozza at and time - including during a game - by using the Strength menu.

The Analyse button open a new web page allowing you to analyse the current position. Both web page will be active.

Optional URL parameters

c=b

You play black. Default is you play white.

l=1 to 10

Sets the initial strength. Default is level 1 (l=1)

fen=fen

Use the specified FEN string to start the game from. The default is the standard start position.

m=ucigocommand

Lozza uses the UCI go command to make her move rather than the fixed time. For example to ask Lozza to search to a depth of 2 ply only use:-

https://op12no2.github.io/lozza-ui/play.htm?m=go depth 2

Another example, to ask Lozza to make her move after searching 100000 nodes, use:-

https://op12no2.github.io/lozza-ui/play.htm?m=go nodes 100000

This can also be used to play with a fixed move time of less than 1 second, for example to make Lozza move after 200ms use:-

https://op12no2.github.io/lozza-ui/play.htm?m=go movetime 200
